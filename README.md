# 社内AI勉強会 告知ページ

社内AI勉強会（Vol.1）の告知用ランディングページです。
GitHub Pagesを利用してホスティングすることを想定しています。

## 🛠 特徴
* **Modern Design:** ダークモードベースのモダンなUI
* **Responsive:** PC、スマホ両対応
* **No Build Required:** HTMLファイル単独で動作（Tailwind CSS CDN利用）

## 🚀 デプロイ方法（GitHub Pages）

1.  このリポジトリを作成し、`index.html` をアップロードします。
2.  リポジトリの **Settings** > **Pages** を開きます。
3.  **Build and deployment** の Source を `Deploy from a branch` に設定します。
4.  Branch を `main` (または `master`) / `/ (root)` に設定し、Saveします。
5.  数分後、表示されたURLにアクセスするとページが公開されます。

## ✏️ 編集方法

`index.html` を直接編集してください。

* **日程・場所の変更:** `` セクション内のテキストを書き換えてください。
* **アジェンダの変更:** `` セクション内を編集してください。
* **リンクの変更:** 「参加登録する」ボタンの `href="#"` を実際のGoogleフォーム等のURLに書き換えてください。

## 🎨 デザイン調整

* **Tailwind CSS** を使用しています。
* クラス名を変更するだけでデザインの微調整が可能です（例: `bg-indigo-600` → `bg-red-600`）。
